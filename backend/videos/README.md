# videos/

Generated lecture MP4 files are written here at runtime by the backend.

This folder is kept in git so the path exists, but actual generated `.mp4` files
are ignored via the root `.gitignore`.

Files kept:
- `dummy_lecture.mp4` (optional placeholder you may remove)
- `fallback_lecture.mp4` (optional placeholder you may remove)

You can safely delete placeholder media before publishing if size matters.
